This Project contains integrated Dashboard of all the below functionalities done by using Python, OpenCV:
1. QR Code Detection and Decoding: Developed a real-time QR code detection and decoding system using OpenCV. 
2. OCR-Based Text Extraction: Built an OCR solution using OpenCV and Tesseract for automated text extraction from images and documents. 
3. Template Matching for Industrial Inspection: Implemented template matching techniques for object verification and quality inspection applications. 
4. Bottle Label Detection: Developed a computer vision solution for detecting and validating bottle labels in images. 
5. Object Counting System: Designed an image-processing application for automated object detection and counting.
